About This Project

WatchaScore is a free, open-source football scoreboard overlay designed for streamers and amateur broadcasters. Customize team names, scores, and the clock in real-time.

I am continuously working to improve WatchaScore. Your feedback and suggestions are highly valued!

If you find this tool useful, please consider supporting its development. Every contribution helps keep the project alive and growing. Don't feel obliged to pay. Use it freely for as long as you like.

Quick Help
Keyboard Shortcuts
[Space] Start / Pause Clock
[C] Hide / Show Clock
[H] Increment Home Goal
[Shift] + [H] Decrement Home Goal
[A] Increment Away Goal
[Shift] + [A] Decrement Away Goal
[Shift] + [X] Reset All
How to use
OBS / Stream
1. Copy this page's URL.
2. Add a Browser Source in OBS and paste the link.
3. Set size to 800 x 600 or higher for better resolution.
4. Crop using the Transform feature.
5. Use Interact to control live.
